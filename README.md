## Kaskádové štýly CSS
### Bohatí vs chudobní - Vitajte v klube 99%
```css
.bohati-ludia {
  top: 1%;
}

.pracujuca-trieda {
  top: 99%;
}
```

### Ako sa správať ku svojej žene - Pravidlo č.1 manželského života
```css
#wife {
  right: 100%;
  margin: 0;
}

```
